# tcc-ilum-piper-chagas
Repositório destinado ao projeto final da disciplina de Iniciação à Pesquisa V da Ilum Escola de Ciência. Título:  "AVALIAÇÃO DE ATIVIDADE ANTI-_TRYPANOSOMA CRUZI_ E CITOTOXICIDADE EM CÉLULAS DE MAMÍFEROS (H9c2) DE PRODUTOS NATURAIS DE _PIPER_ SPP. E SINTÉTICOS _IN VITRO_"

Neste repositório foram depositados dados brutos obtidos dos ensaios fenotípicos de epimastigotas e amastigotas, além daqueles referentes a etapas intermediárias. Os arquivos utilizados para tratamento e processamentos dos dados estarão disponíveis futuramente.
